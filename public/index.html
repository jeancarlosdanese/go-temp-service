API de Busca de Temperatura por CEP

Esta aplicação busca a localização de um CEP e retorna a temperatura atual da cidade em Celsius, Fahrenheit e Kelvin.

Como Utilizar

Envie uma solicitação GET para a rota /weather?cep=SEU_CEP com um CEP válido de 8 dígitos.

Exemplo: /weather?cep=89500001

Exemplo de Resposta

{
  "temp_C": 19.2,
  "temp_F": 66.56,
  "temp_K": 292.35
}

Tratamento de Erros

  • CEP inválido: Retorna status 422 com a mensagem {"message": "invalid zipcode"}
  • CEP não encontrado: Retorna status 404 com a mensagem {"message": "can not find zipcode"}
  • Erro ao buscar dados de clima: Retorna status 500 com a mensagem {"message": "error fetching weather data"}

Sobre

Este projeto foi desenvolvido em Go e utiliza Docker para containerização e Google Cloud Run para deploy.

Consulte o repositório no GitHub para mais detalhes.
